Some projects I've worked on...

2048

2048 was a class project for Java II. Several of us contributed towards an end goal of remaking the popular game, and for many of us it was our first time programming in a group setting. Coordinating and getting everything to work was a learning experience for all of us.

My segment was the "combine" part of the game, where it decides whether or not two tiles can combine together.

Portfolio v2.0

The second life of my portfolio website - the one you are looking at right now! I experimented more with flexboxes, columns and font icons in this version.

The Silver Project

I've had a hard time staying focused on my goals, so I started this project as an attempt to motivate myself to learn more. Soon, I will commit to the 100 days of code challenge, and document my progress through blog posts, blurbs, and a special Github repository.

This way, I hope to stay motivated, and to accelerate my learning. My primary focus language will be Java, although I'm sure there will be bits and pieces of other langauges and tools thrown in.
